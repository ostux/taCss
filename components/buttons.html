Effects

The buttons effects can be .raised and .ripple. The ripple color same as the current text color the given element (oppacity 0.5). Also the ripple effect need overflow: hidden... It's mean, you can't use with bages or any other element which need overflow: visible.

Size

Color

Button can be colored as well. You can use any color from the color palette. To set the text color readable automaticaly use the colored class within your choosen background color. Use the color name with the .bg- prefix to set the background color. Colorize just the text, use the .text- prefix. To set the outline color and the text then use the .outline- prefix with your choosen color. Of course you can use with normal buttons or FAB as well... like any other options.

Disabled

Disabled the button by using the disabled attribute. Use the disabled class with links. Ripple effect and colors are not effecting the disabled buttons.

Width

Sometime need a button to take all aviable space. Well then use the .btn-block class to achive this. The button will grow up all available space horizontaly. The button height not affected. Don't use .ripple efect with full lenght button because the ripple effect diameter 200px, buttons wider than 200px look not nice.
The button with possible to setup using the .col-X classes as well. The gutter still effecting the button with. If you don't want gutter use the no-gutter class.
