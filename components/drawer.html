taCss at the moment only one drawer object can handle per page and this drawer is in the left side. Howewer if you need more or in the other site, just grab the source and alter it as you wish. (I don't need more, I don't need in the other side and no one else use this so why the effort...)

Drawer shoud be the direct children of the body. Preferable the first. (Or the second if you want to use full width navbar).

fixed, not fixed

Drawer can be fixed or not fixed. It's mean always (all screen size) visible or need to open. Howewer the drawer responsive. On small screen automatic close itself even if is fixed! Small screen not equal to mobile screen size! Small screen breakpoint is half way of the tablet. [(desktop breakpoint - mobile breakpoint) / 2 + mobile breakpoint]
To fix the drawer just use the drawer-fixed class within the drawer.
