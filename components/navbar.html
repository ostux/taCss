fixed navbar

The fixed navbar come with the feature like the navbar always on top of the page, and with lot of headache. Because the fixed navbar out of the normal flow (position: fixed) to take care the rest of the document a bit tricky specialy without using any javascript. taCSS prefering the CSS way. Using javascript if is necessery or way easier then using pure CSS. You can use as many non fixed navbar as you wish. Howewer this is not true with the fixed navbar. You can use maximum 3 with the basic setup. Of course you can increase or even decrease this in the variables file. Look for the $navbar-max-fixed-line: 3 !default setting.

shadow

If you have just non fixed navbar or navbars use the shadow-X class (X = 2, 3, 4, 6, 8, 16, 24) within the header element. If you have only one fixed navbar you can use the shadow-X class within the navbar. In this case do not use with the header.

If you have more than one fixed navbar than you have to use the <div class='navbar-fixed-shadow shadow-X'></div> element. This element must be the last nav element. If you have one ore more fixed and one or more not fixed navbar than you have to use the <div class='navbar-fixed-shadow shadow-X'></div> element after all navbar (include the not fixed as well) and use the shadow-X on the header element.

If you have MORE THAN ONE and ONLY fixed navbars, don't forget to setup margin-top for the first element after the header in your css file.
Navbar sizes and breakpoint settings you can found in the _variables.sass file. {% highlight scss %} $breakpoint-desktop: (d, 12, 900) !default $navbar-height: 56px !default $navbar-height-desktop: 64px !default {% endhighlight %}
{% highlight css %} /*-- add this code for your css file */ .firstElementAfterHeader { margin-top: calc(56px * howManyFixedNavbarYouHave); } @media only screen and (min-width: 900px) { .firstElementAfterHeader { margin-top: calc(64px * howManyFixedNavbarYouHave); } } {% endhighlight %}
dropdown in the navbar

Use dropdown in the header as well if you want. Don't use dropdown menu on fixed navbar if the next navbar fixed as well!
