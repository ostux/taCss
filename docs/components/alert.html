About
Alerts

Provide contextual feedback messages for typical user actions with flexible alert messages.

Examples

Alerts are available for any length of text, as well as an optional dismiss button. For styling, you can use one of the five predefinied classes ( .primary, .success, .info, .warning, .danger)
To use the predefinied classes require to set $enable-variants: true !default in base/_variations.sass!

Hello! This is the basic default alert.
Well... You read this primary alert message.
Well done! You successfully read this important alert message.
Heads up! This alert needs your attention, but it's not super important.
Warning! Better check yourself, you're not looking too good.
Oh snap! Change a few things up and try submitting again.
