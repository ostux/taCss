Color Palette
Background

Use the bg- prefix with your color of choice.

Auto textcolor

To choose readable font color automatic to the choosen background use the colored class.

Outline (border and text color)

Use the outline- prefix with your color of choice to get colored border and text.

Text color

Use the text- prefix with your color of choice to get colored text.

The full color palette

You can chose which color you want to include by seting true or false in the base/_variables.sass the given color palette. The settings look like: $enable--red-palette: true !default The basic color palette contains all base color in each color palette (without number), excluding the accent color. If you do not want to include any of this color, than do not forget to set to false the $enable-colors: true !default settings. Without the color palette the components coloring not working. Howewer the color variants working fine unless you turn off in the settings. If you want to use the component coloring then you have to include minimum two color. One for the primary one for the accent color. You can set up them in the variables file color section. like: $primary-palette: green !default, $accent-palette: pink !default
