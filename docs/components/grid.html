Grid
row / col

To get one row just use the row class. Then your container will be flex container and flex-direction: row.
If you want display in column from top to bottom then use the col class. Then your container will be flex container and flex-direction: column.

column width

To set up the width of the columns are easy...
taCSS contains 12 colums in desktop screen, 8 colums in tablet and 4 colums in mobile screen. Four type of column:

  • col-X, off-X

    Universal columns. From: col-1 to col-12. If you use col-6 then in mobile device the width is 100%, in tablet: 75% and in desktop: 50%. Universal offset. From: off-1 to off-11. Make no sense for off-12...

  • col-p-X, off-p-X

    Phone columns. From col-p-1 to col-p-4. Phone offset. From off-p-1 to off-p-3.

  • col-t-X, off-t-X

    Tablet columns. From col-t-1 to col-t-8. Tablet offset. From off-t-1 to off-t-7.

  • col-d-X, off-d-X

    Desktop columns. From col-d-1 to col-d-12. Desktop offset. From off-d-1 to off-d-11.

off-

The offset use margin-left to push the next content. Make sure you don't add any extra margin, padding or contain even border!
