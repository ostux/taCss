Fixed grid system allows given columns across the page. Dynamic grid allows to have different column number each view. (mobile, tablet, desktop) taCss can be whichever you want.

To have fixed grid just change the settings to the same column number in each view.

You want gutter by default? Not a problem, howewer the default setup for gutter is true! TODO: link for spacing page when done!.

row: flex container with flex-direction: row

col: flex container with flex-direction: column

col- the prefix for grid columns.
off- the prefix for offsets. The offset use margin-left to push the next content. Make sure you don't add any extra margin, padding or contain even border!

no-gutter: force the actual column to be gutter free. (If the gutter are enabled, anyway is useless)

grid setup
{% highlight scss %} /* grid class names in "core/classnames.scss" */ $class-grid-row: "row" !default; $class-grid-column: "col" !default; $class-grid-column-prefix: "col-" !default; $class-frid-off: "off-" !default; $class-grid-no-gutter: "no-gutter" !default; /* grid setups in "core/_variables.scss" */ /* enable or disable gutter */ $enable-gutter: true !default; /* breakpoint name, columns, width by pixel in mobile(end), tablet(start), desktop(start), gutter size */ $breakpoint-mobile: m, 4, 479, .5rem !default; $breakpoint-tablet: t, 8, 480, 1rem !default; $breakpoint-desktop: d, 12, 900, 1.5rem !default; {% endhighlight %}
column widths
  • col-X, off-X

    Universal columns. From: col-1 to col-12.
    You can found the maximum columns setup in the "core/_variables.scss" file in the breakpint-desktop array second element.
    If you use col-6 in mobile device the width is 100% (because in default 4 column in mobile, so 4-12 are 100%), in tablet: 75% and in desktop: 50%. Universal offset. From: off-1 to off-11. Make no sense for off-12...

  • col-p-X, off-p-X

    Phone columns. From col-p-1 to col-p-4. Phone offset. From off-p-1 to off-p-3.

  • col-t-X, off-t-X

    Tablet columns. From col-t-1 to col-t-8. Tablet offset. From off-t-1 to off-t-7.

  • col-d-X, off-d-X

    Desktop columns. From col-d-1 to col-d-12. Desktop offset. From off-d-1 to off-d-11.
